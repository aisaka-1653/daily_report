- JavaScript
  - イベントハンドラの設定
    - プロパティを設定する
      ```javascript
        const = e.document.getElementById('button');
        e.onclick = () => {
          console.log('Clicked!');
        }
      ```
    - loadイベント
      ```javascript
        window.onload = () => {
          console.log('load event');
        }
      ```
    - addEventListener
      ```javascript
        const = e.document.getElementById('button');
        e.addEventListener = ('click', () => {
          console.log('Clicked!');
        }, false);
      ```
